Google Maps Geocoding API Key Test

This will test if your API key is working correctly.

Common Issues & Fixes:

  • REQUEST_DENIED:
    • Check if Geocoding API is enabled in Google Cloud Console
    • Verify API key restrictions allow your domain/localhost
    • Ensure billing is enabled (required even for free tier)
  • API key restrictions: For local testing, add:
    • http://localhost:*
    • http://127.0.0.1:*
    • file://* (if testing with file:// protocol)
